=head1 NAME

EVP_MD-BLAKE3 - The BLAKE3 EVP_MD implementation

=head1 DESCRIPTION

Support for computing BLAKE3 digests through the B<EVP_MD> API.

=head2 Identities

This implementation is only available with the default provider, and includes
the following varieties:

=over 4

=item BLAKE3-256

Known names are "BLAKE3" and "BLAKE3-256".

=back

=head2 Gettable Parameters

This implementation supports the common gettable parameters described in
L<EVP_MD-common(7)>.

=head1 SEE ALSO

L<provider-digest(7)>, L<OSSL_PROVIDER-default(7)>, L<EVP_MD-common(7)>,
L<EVP_DigestInit(3)>
